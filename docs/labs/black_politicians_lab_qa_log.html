5 QA Decision

The current evaluated code is training-aligned with The Effect methods and dataset framing, but it is not book-identical in specification.

If the goal is strict replication of the chapter’s black_politicians workflow, two updates are required:

  1. Add a “book-default spec” mode that uses only medianhhincom, blackpercent, and leg_democrat for CEM/entropy balancing.
  2. Add the weighted interaction model with treat_out and substantive controls as a standard evaluated output block.
